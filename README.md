# 🌊 Rising Waters - Flood Prediction System

A Machine Learning web application that predicts flood risk using weather and environmental parameters. The application is built with **Flask** and powered by an **XGBoost** model to provide accurate flood risk predictions through a simple and interactive web interface.

## 🚀 Live Demo

🔗 **Website:** https://flood-prediction-v8fs.onrender.com

## Clone the repository:

```bash
git clone https://github.com/Vamc-24/Rising-Waters.git
cd Rising-Waters
```

## ✨ Features

- 🌧️ Predict flood risk using Machine Learning
- 🤖 XGBoost-based prediction model
- 🌐 Interactive Flask web application
- 📱 Responsive and user-friendly interface
- ⚡ Instant flood risk prediction
- 📊 Displays prediction confidence and model accuracy

## 🛠️ Technologies Used

- Python
- Flask
- XGBoost
- NumPy
- Pandas
- Scikit-learn
- HTML5
- CSS3

## 📊 Input Features

The prediction model uses the following weather parameters:

- 🌡️ Temperature
- 💧 Humidity
- ☁️ Cloud Cover
- 🌧️ Annual Rainfall
- 📅 Average June Rainfall
- 🌍 Sub-region Rainfall

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Vamc-24/Rising-Waters.git
cd Rising-Waters
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python app.py
```

### 4. Open in your browser

```
http://127.0.0.1:5000
```
